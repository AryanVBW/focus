# Installation Guide

This guide will walk you through the process of installing Focus on your Android device and configuring it for optimal performance.

## System Requirements

- **Android Version**: 7.0 (API level 24) or higher
- **Storage**: At least 50MB of free space
- **RAM**: Minimum 2GB recommended for smooth operation

## Download Focus

1. Visit our [GitHub Releases page](https://github.com/AryanVBW/focus/releases)
2. Download the latest APK file
3. Ensure you're downloading from the official repository to avoid security risks

## Installation Steps

### Step 1: Enable Unknown Sources

Before installing the APK, you need to allow installation from unknown sources:

1. Go to **Settings** > **Security** (or **Privacy**)
2. Enable **Unknown Sources** or **Install unknown apps**
3. Select your browser or file manager and allow it to install apps

### Step 2: Install the APK

1. Open the downloaded APK file
2. Tap **Install** when prompted
3. Wait for the installation to complete
4. Tap **Open** to launch Focus

## Initial Setup

### Required Permissions

Focus requires several permissions to function properly:

#### Accessibility Service
- **Purpose**: Detect and block distracting content
- **Setup**: Settings > Accessibility > Focus > Enable
- **Note**: This is the core permission that enables Focus to work

#### Usage Stats Permission
- **Purpose**: Track app usage time (optional but recommended)
- **Setup**: Settings > Apps > Special access > Usage access > Focus > Enable

#### Display Over Other Apps
- **Purpose**: Show blocking pages when distractions are detected
- **Setup**: Settings > Apps > Special access > Display over other apps > Focus > Enable

#### Query All Packages
- **Purpose**: Show list of installed apps for selection
- **Setup**: Automatically requested during first launch

### Configuration

1. **Choose Apps to Monitor**: Select which apps you want Focus to monitor
2. **Set Blocking Preferences**: Configure what content to block in each app
3. **Configure Focus Mode**: Set up timers and blocking levels
4. **Customize Notifications**: Choose how you want to be notified

## Verification

To verify that Focus is working correctly:

1. Open a monitored app (like Instagram)
2. Try to access blocked content (like Reels)
3. You should see a Focus blocking screen
4. Check the usage statistics in the Focus app

## Troubleshooting Installation

### Common Issues

**APK won't install**
- Ensure you have enough storage space
- Check that unknown sources are enabled
- Try redownloading the APK file

**Permissions not working**
- Restart your device after granting permissions
- Check that all required permissions are enabled
- Some devices may require additional steps for accessibility services

**App crashes on startup**
- Clear the app's cache and data
- Restart your device
- Ensure your Android version is supported
